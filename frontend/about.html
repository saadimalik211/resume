About This Site

This site is built from scratch, hosted and running on AWS.

  • HTML, CSS, and JavaScript for frontend development
  • Python for backend development
  • Git for version control
  • GitHub Actions for frontend CI/CD into AWS
  • Amazon S3 for hosting
  • Amazon CloudFront for content delivery and access control
  • Amazon Route 53 for DNS management
  • Amazon ACM for SSL certificate management
  • AWS Lambda for the visitor counter feature
  • AWS DynamoDB for visitor count storage
  • AWS HTTP API Gateway for serverless functionality
